# 需求管理平台 · 完整功能清单

> 版本 v1.0 | 生成时间 2026-05-28

---

## 一、系统概览

这是一个 **AI 驱动的需求管理平台**，面向两类用户：
1. **PM（产品经理）**：通过管理后台（Dashboard）统一管理来自多个国家大区和 SaaS 厂商的需求
2. **大区 / SaaS 厂商**：通过专属链接提交需求、追踪状态

核心流程：**需求提交 → AI 自动归并 → 优先级评分 → 状态追踪**

## 四、需求提交页 — 大区版（`form.html`）

### 4.1 访问控制

- 通过 URL 参数 `?token=xxx` 验证身份
- Token 无效时显示「链接无效」页面
- 自动识别大区信息（名称、国旗、语言）

### 4.2 多语言支持

| 语言 | 说明 |
|------|------|
| EN | English |
| 中文 | 简体中文 |
| IT | Italiano |
| FR | Français |
| ภาษาไทย | 泰语（自动加载 Noto Sans Thai 字体） |
| ខ្មែរ | 高棉语（自动加载 Noto Sans Khmer 字体） |

### 4.3 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 需求描述 | 多行文本框 | ✅ | 支持本地语言填写 |
| 来源商户 | 文本框 | ✅ | 商户名称或编号 |
| 商户类型 | 单选 | ✅ | 餐厅 / 零售 / 其他 |
| 业务影响 | 分段控件 | ✅ | 🔴高影响（阻塞核心）/ 🟡中影响 / 🟢低影响 |
| 涉及场景 | 多选 | ✅ | POS / 报表 / 会员 / 厨显 / 支付 / 库存 |
| 附件 | 文件上传 | ❌ | 图片/视频，单文件最大 20MB，支持拖拽上传 |

### 4.4 自动记录字段

| 字段 | 说明 |
|------|------|
| 来源国/大区 | 自动从 Token 解析 |
| 提交时间 | 实时显示 |
| 记录 ID | 自动生成预览 |

### 4.5 提交流程

1. 填写表单 → 点击「提交需求」
2. 调用 `POST /api/tickets` 接口
3. 显示成功弹窗：工单 ID + AI 处理提示
4. 可点击「继续提交」重置表单

### 4.6 新手引导

- 首次访问自动弹出 2 步引导浮层
- 支持切换引导语言
- 可勾选「不再提醒」（localStorage 存储）

| `cluster_tickets` | 需求簇 ↔ 工单关联表 |
| `scoring_config` | 评分规则配置（JSON 格式） |
| `saas_vendors` | SaaS 厂商信息（token、名称、行业、代码） |
| `saas_tickets` | SaaS 工单 |
| `saas_clusters` | SaaS 需求簇 |
| `saas_cluster_tickets` | SaaS 需求簇 ↔ 工单关联表 |
